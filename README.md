公孙离技能库（Cursor Skill）

基于官方 CG 资源的 Cursor 技能集成仓库

### 技能导入（Cursor 端）

1.打开 Cursor 编辑器，进入「规则与技能（Rules, Skills, Subagents）」配置页

2.选择「添加远程技能（Remote Skill）」，粘贴本仓库地址：https://github.com/hehuan78/gongsunli-skill.git

3.确认添加后，重启 Cursor 即可加载技能，支持直接调用技能相关指令

### 调用规范
技能加载后，可通过 Cursor 内置指令直接调用（例如 /gongsunli 相关指令）

遵循官方 CG 定义的技能交互规则，禁止违规修改核心逻辑
