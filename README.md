公孙离技能库（Cursor Skill）
基于官方 CG 资源的 Cursor 技能集成仓库
适配 Cursor 编辑器的技能配置仓库，提供官方 CG 风格的技能定义与使用说明，支持快速集成与版本迭代。
📌 仓库概览
本仓库用于管理 公孙离技能库（gongsunli-skill） 的核心配置文件 SKILL.md，遵循 Cursor 技能配置规范，基于官方 CG 资源构建，可直接用于 Cursor 编辑器的技能加载与调用，支持技能更新、版本管理与场景适配。
🎯 核心功能
官方 CG 技能适配：完全对齐官方 CG 设定的技能逻辑与交互规则，还原官方技能体验。
快速集成：支持 Cursor 一键导入、本地编译与云端同步，无需复杂配置。
版本可控：记录技能迭代节点，支持版本回滚、特性更新，适配不同场景需求。
场景化适配：覆盖技能查询、指令调用、参数配置等核心使用场景，满足日常开发与调试需求。
🚀 快速开始
1. 技能导入（Cursor 端）
打开 Cursor 编辑器，进入「规则与技能（Rules, Skills, Subagents）」配置页。
选择「添加远程技能（Remote Skill）」，粘贴本仓库地址：https://github.com/hehuan78/gongsunli-skill.git。
确认添加后，重启 Cursor 即可加载技能，支持直接调用技能相关指令。
2. 本地更新与同步
若需修改技能配置或同步最新内容，可通过以下命令操作：
powershell
# 进入本地技能目录
cd C:\Users\xxx\Cursor\skills\gongsunli-skill

# 拉取最新更新
git pull

# 若需自定义更新，修改配置后执行提交
git add . && git commit -m "技能更新描述" && git push
